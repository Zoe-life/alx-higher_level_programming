# Python If/Else, Loops, and Functions

## Overview

Exercises focused on Python control flow, arithmetic, ASCII output, looping patterns, and reusable helper functions.

## Learning Objectives

- Apply conditional logic with `if`, `elif`, and `else`.
- Use `for` loops for character and number sequences.
- Write and call small utility functions.
- Practice simple algorithmic thinking with numeric exercises.
