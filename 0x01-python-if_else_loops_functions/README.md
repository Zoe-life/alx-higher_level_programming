# Python If/Else, Loops, and Functions

## Overview

Exercises focused on Python control flow, arithmetic, ASCII output, looping patterns, and reusable helper functions.

## Learning Objectives

- Apply conditional logic with `if`, `elif`, and `else`.
- Use `for` loops for character and number sequences.
- Write and call small utility functions.
- Practice simple algorithmic thinking with numeric exercises.

- Python 3
- pycodestyle

## Usage

Execute scripts with `python3 <filename>.py`. The included `*-main.py` files can be used to validate expected behavior during local testing.

## Detailed File Guide

- `0-positive_or_negative.py` — Prints whether a random number is positive, zero, or negative.
- `1-last_digit.py` — Extracts and classifies the last digit of a random number.
- `2-print_alphabet.py` — Prints lowercase alphabet in order.
- `3-print_alphabt.py` — Prints alphabet except `q` and `e`.
- `4-print_hexa.py` — Prints numbers 0-98 in decimal and hexadecimal.
- `5-print_comb2.py` — Prints numbers 00-99 with formatting.
- `6-print_comb3.py` — Prints all different two-digit combinations.
- `7-islower.py` — Returns `True` if a character is lowercase.
- `8-uppercase.py` — Prints a string in uppercase without `str.upper()`.
- `9-print_last_digit.py` — Returns and prints the last digit of a number.
- `10-add.py` — Adds two integers and returns the result.
- `11-pow.py` — Computes `a` to the power `b`.
- `12-fizzbuzz.py` — Prints the classic FizzBuzz sequence from 1 to 100.
- `100-print_tebahpla.py` — Prints alphabet in reverse, alternating case.
- `101-remove_char_at.py` — Returns a string with a character removed at index `n`.
- `102-magic_calculation.py` — Recreates specified bytecode behavior in Python source.
- `linked_lists.c` — Linked-list helper function used by an advanced task.
- `lists.h` — Header definition for linked-list utilities.
