# JavaScript Warm Up

## Overview

Introductory JavaScript exercises for Node.js covering variables, loops, functions, arithmetic, and command-line argument parsing.

## Learning Objectives

- Write and execute basic Node.js scripts.
- Handle command-line input and numeric conversion.
- Practice loops, functions, and object usage in JavaScript.
- Build confidence with concise scripting exercises.

## Requirements

- Node.js

## Usage

Run scripts with `node <filename>.js` from within the project directory.

## Detailed File Guide

- `0-javascript_is_amazing.js` — Prints "JavaScript is amazing" using a constant.
- `1-multi_languages.js` — Prints three lines using `console.log`.
- `2-arguments.js` — Prints message based on argument count.
- `3-value_argument.js` — Prints first argument or fallback text.
- `4-concat.js` — Concatenates and prints two CLI arguments.
- `5-to_integer.js` — Converts first argument to integer if possible.
- `6-multi_languages_loop.js` — Prints predefined lines using loop iteration.
- `7-multi_c.js` — Prints `C is fun` multiple times from numeric argument.
- `8-square.js` — Prints square made of `X` characters.
- `9-add.js` — Adds two integers from CLI args.
- `10-factorial.js` — Computes factorial recursively from CLI input.
- `11-second_biggest.js` — Prints second largest integer from CLI args.
- `12-object.js` — Updates object value and prints before/after states.
- `13-add.js` — Exports reusable `add(a, b)` function.
- `100-let_me_const.js` — Modifies imported variable via assignment.
- `101-call_me_moby.js` — Exports function that repeats callback execution.
- `102-add_me_maybe.js` — Exports function that increments index before callback.
- `103-object_fct.js` — Imports object and adds function-based behavior.
