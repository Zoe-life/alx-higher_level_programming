# Python Classes

## Overview

An introduction to object-oriented programming in Python through progressively richer class definitions built around a square abstraction.

## Learning Objectives

- Define classes with constructors and instance attributes.
- Apply encapsulation and validation rules.
- Implement methods for behavior and representation.
- Model linked structures and derived behavior in Python classes.

Run examples with `python3 <filename>.py` or use the supplied `*-main.py` files to exercise each milestone implementation.
