## Usage

Run scripts with `python3 <filename>.py` exactly as you would in the main `0x04-python-more_data_structures` directory.

## Detailed File Guide

- `0-square_matrix_simple.py` — Returns matrix with each value squared.
- `1-search_replace.py` — Replaces all occurrences of one value in a list.
- `2-uniq_add.py` — Sums unique integers in a list.
- `3-common_elements.py` — Returns set intersection between two sets.
- `4-only_diff_elements.py` — Returns symmetric difference of two sets.
- `5-number_keys.py` — Counts keys in a dictionary.
- `6-print_sorted_dictionary.py` — Prints dictionary items sorted by key.
- `7-update_dictionary.py` — Updates/adds key-value pair in dictionary.
- `8-simple_delete.py` — Deletes a key from dictionary if present.
- `9-multiply_by_2.py` — Returns new dict with all values multiplied by 2.
- `10-best_score.py` — Finds key with highest integer value.
- `11-multiply_list_map.py` — Multiplies list values using `map`.
- `100-weight_average.py` — Computes weighted average from tuple list.
- `101-square_matrix_map.py` — Squares matrix values with `map` + `lambda`.
- `102-complex_delete.py` — Deletes keys from dictionary when value matches target.
- `103-python.c` — C utilities that inspect Python string/list internals.
- `103-tests.py` — Python test driver for `103-python.c` helpers.
