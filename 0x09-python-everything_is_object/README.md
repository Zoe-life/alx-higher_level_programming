# Python Everything Is Object

## Overview

Conceptual exercises and short scripts that explore identity, mutability, references, assignment behavior, and Python object semantics.

## Learning Objectives

- Reason about object identity and equality.
- Understand mutable versus immutable behavior.
- Analyze assignment, aliasing, and side effects.
- Practice concise proof-of-concept scripts for object model questions.
