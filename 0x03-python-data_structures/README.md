# Python Data Structures

## Overview

Exercises exploring lists, tuples, indexing, mutation, matrix-style iteration, and sequence-related helper functions.

## Learning Objectives

- Manipulate lists and tuples safely and efficiently.
- Access and replace elements by index.
- Iterate through nested list structures.
- Implement helper functions for common sequence operations.

- pycodestyle

## Usage

Use `python3 <filename>.py` for the Python deliverables. Compile the C file with the matching header when testing the linked-list related exercise.
