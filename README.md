# ALX Higher Level Programming

This repository documents and organizes a collection of ALX higher-level programming projects completed across Python, JavaScript, SQL, shell scripting, and object-oriented programming modules. Each project folder contains focused exercises that reinforce core language features, problem-solving, testing, and software design fundamentals.

## Repository Overview

The work in this repository follows a project-based learning path that progresses from Python basics to object-oriented design, data structures, file handling, networking, databases, and introductory JavaScript.

## Common Technologies

- **Python 3** for scripting, algorithms, and object-oriented programming exercises
- **JavaScript (Node.js)** for introductory backend scripting tasks
- **Shell scripting** with `bash` and `curl` for networking exercises
- **SQL / MySQL** and **SQLAlchemy** for relational database work
- **C** for selected interoperability and low-level exercises included in some Python projects

## Working With the Repository

1. Open any project folder listed above.
2. Read the local `README.md` for project-specific context and requirements.
3. Run the relevant scripts with the appropriate interpreter (`python3`, `node`, `bash`, or MySQL).
4. Use the provided `*_main.py` and `*_main.js` files as local drivers where available.
