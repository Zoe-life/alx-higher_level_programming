# Python More Classes

## Overview

A continuation of object-oriented practice focusing on rectangles, class attributes, static methods, class methods, and algorithmic problem solving.

## Learning Objectives

- Extend class-based design beyond introductory examples.
- Use class attributes and alternate constructors effectively.
- Implement string representations and comparison helpers.
- Solve the N-Queens challenge with clean Python structure.
