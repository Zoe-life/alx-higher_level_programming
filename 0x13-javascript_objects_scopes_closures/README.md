# JavaScript Objects, Scopes, and Closures

## Overview

A follow-on JavaScript project focused on object construction, inheritance-style patterns, closures, array transformations, and file concatenation utilities.

## Learning Objectives

- Create and extend objects with constructor functions or classes.
- Understand scope and closure behavior in JavaScript.
- Transform arrays and datasets with reusable functions.
- Work with filesystem-style scripting inputs and outputs.

## Usage

Execute each deliverable with `node <filename>.js`. Use the bundled `*-main.js` files for quick local checks where provided.

## Detailed File Guide

- `0-rectangle.js` — Defines empty rectangle class shell.
- `1-rectangle.js` — Rectangle class with width and height constructor.
- `2-rectangle.js` — Skips object creation when dimensions are invalid.
- `3-rectangle.js` — Adds instance method to print rectangle with `X`.
- `4-rectangle.js` — Adds rotate and double size methods.
- `5-square.js` — Square class inheriting from rectangle.
- `6-square.js` — Adds char-based print method for square.
- `7-occurrences.js` — Counts occurrences of value in array.
- `8-esrever.js` — Returns reversed list without using built-in reverse.
- `9-logme.js` — Closure-based logger with incrementing call count.
- `10-converter.js` — Returns closure that converts numbers to selected base.
- `100-map.js` — Creates mapped array multiplied by index.
- `100-data.js` — Data source module used by `100-map.js`.
- `101-sorted.js` — Groups dictionary keys by occurrence count values.
- `101-data.js` — Data source module for sorted-occurrence task.
- `102-concat.js` — Concatenates two files into a third file.
- `fileA`, `fileB`, `fileC` — Sample files for file concatenation testing.
